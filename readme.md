# Contact Book (Python CLI)

A command-line contact management application built using Python.  
The app allows users to add, view, search, and delete contacts with automatic JSON-based persistence.

## Features
- Add new contacts
- View all contacts
- Search contacts by name
- Delete contacts
- Automatic save to JSON file
- Input validation for phone number and email

## Technologies Used
- Python
- JSON
- File Handling
- CLI-based menu system

## How to Run
```bash
python contact_book.py
